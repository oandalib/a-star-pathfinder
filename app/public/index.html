A* Pathfinder

  1. Choose start and end position
    You may have to slightly wiggle the mouse when you click, this allows later for dragging your mouse for barriers
  2. Choose barrier positions
  3. You may remove cells with the middle mouse button
  4. Click on the Find Path button
  • Yellow is the start position
  • Blue is the end position
  • Black indicates barrier
  • Green indicates cell has been added to open set (to be considered)
  • Red indicates cell has been added to closed set
  • Purples indicates the path found
